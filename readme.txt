OMIT (Opinion Mining In Teaching)

3 de julio de 2013

Proyecto de Fin de Grado de Jónatan Núñez para la Universidad Rey Juan Carlos de Madrid en colaboración con Soto Montalvo.

En la carpeta "docs" se encuentra la memoria del proyecto así como un manual técnico de configuración.

El proyecto ha sido probado con:
	· Apache 2.4
	· MySQL 5.5
	· PHP 5.4
	· Solr 4.3

La estructura de la base de datos de MySQL encuentra en /init/estructura.sql
La estructura de la base de datos de Lucene se encuentra en la carpeta /solr/
